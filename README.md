# MyFridge 🧊

A highly visual productivity and organization app inspired by a real refrigerator door.

MyFridge transforms the browser new-tab experience into a digital fridge surface where users can place sticky notes, reminders, photos, links, checklists, drawings, and decorative magnets — just like a real fridge door.

The experience is designed to feel physical, fast, playful, and naturally organized.

## ✨ Features

- 📝 Sticky notes and reminders
- ✅ Checklist and grocery tracking
- 📌 Drag-and-drop fridge surface
- 📷 Photo and memory cards
- 🔗 Smart link and info cards
- 🎨 Decorative stickers and magnets
- ⚡ Quick item capture and editing
- 🧘 Focus mode for important items
- 🤖 Fridge Assistant for organization
- ⚙️ Multiple fridge themes and customization
- 📱 Fully responsive design


## 💡 Product Idea

People naturally use refrigerator doors to manage everyday life:
- grocery lists
- reminders
- bills
- family notes
- schedules
- photos and memories

MyFridge recreates that familiar experience digitally with a realistic fridge-door interface and freeform interactions.

### Experience Goal

Open → Glance → Act → Leave

Fast, visual, and distraction-free.


## 🛠️ Tech Stack

- React
- Vite

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/sajanbodele/MyFridge.git
```

Install dependencies:

```bash
npm install
```

Create an `.env.local` file:

```env
VITE_BASE44_APP_ID=your_app_id
VITE_BASE44_APP_BASE_URL=your_backend_url
```

Run the development server:

```bash
npm run dev
```
